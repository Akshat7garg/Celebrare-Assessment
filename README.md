# 📸 Pixora

A modern and minimal **photo gallery application** built with **React + Vite**, designed for smooth browsing, clean UI, and fast performance. Explore high-quality images, mark favourites, and enjoy seamless infinite scrolling.

---

[![Screenshot](https://pixora-tan.vercel.app/preview.png)](https://pixora-tan.vercel.app/)

## 🌐 Live Preview

> <https://pixora-tan.vercel.app/>

---

### ✨ Features

🖼 Explore high-quality images
♾ Infinite scrolling gallery
⭐ Add and remove favourite photos
🌗 Light / Dark theme support
⚡ Fast and responsive UI
🧠 State management using React Hooks and Reducer

---

### ⚙️ Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/Celebrare-Assessment.git
cd Celebrare-Assessment

# Install dependencies

npm run dev
```
